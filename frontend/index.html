Input Guide

You can either input your data directly or upload a text file, where each line denotes a different instruction

The following are the rules for a valid input for each line

  1. Each line should be in the format:
    • YYYY-MM-DD HH:MM USER_ONE recommends USER_TWO
    • YYYY-MM-DD HH:MM USER_ONE accepts

    where YYYY-MM-DD is the year-month-day format and HH:MM is a 24-hour format, USER_ONE and USER_TWO should be a word only character with no space or special characters

  2. All inputs are case insensitive, this means that user "A" is the same as user "a", "accepts" is the same as "Accepts" and "ACCEPTS", etc
